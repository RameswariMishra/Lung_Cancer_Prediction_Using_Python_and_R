# Notebooks

This folder contains Jupyter notebooks implementing the end-to-end machine learning workflow for lung cancer risk prediction using both Python and R.

## Contents

- **Lung_Cancer_Prediction_Using_Python_And_R.ipynb**  
  Performs data preprocessing, exploratory data analysis (EDA), feature analysis, model development, and evaluation using machine learning techniques implemented in both Python and R. The notebook compares predictive performance and interpretability across the two analytical ecosystems.
